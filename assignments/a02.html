Introduction

This Quarto document follows the Week 5 Assignment 2 practice with Tidy Data. This document will explore whether a data set I have follows the Tidy Data principles. To recap, the three Tidy Data principles are as follows:

  1. Each column has is one variable

  2. Each row is one observation.

  3. Each cell is one value.

About the data used

I have been culturing Human umbilical cord blood derived erythroid progenitor cells over the past month or so. I have been seeding them at a density of 100,000 cells/mL, and over the course of culturing these cells, I have been recording the number of cells in my total culture to evaluate how well they are expanding. This data is compiled in a data file titled “BIOL806_cell_counts_data.csv”.

Read in the data

Quarto runs R in the same way as a script: objects only exist after the chunks that create them have been run.
